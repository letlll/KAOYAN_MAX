HTML 表单属性与类型

1. `
` 标签的 `method` 属性可以取哪些属性值？分别表示什么含义？

  • `GET`：通过 URL 传递表单数据，数据附加在 URL 后，适合请求数据。数据量有限，不能包含敏感信息。
  • `POST`：通过 HTTP 请求体发送表单数据，数据不显示在 URL 中，适合提交敏感信息或较大数据。

2. HTML5 中新增了哪些 `` 标签的常用类型？分别表示什么含义？

  • `email`：用于输入电子邮件地址，浏览器可进行基本的格式验证。
  • `url`：用于输入 URL 地址，浏览器会验证格式。
  • `tel`：用于输入电话号码，浏览器提供适当的键盘布局（移动设备上）。
  • `date`：用于输入日期，显示日期选择器（根据浏览器支持）。
  • `time`：用于输入时间，显示时间选择器（根据浏览器支持）。
  • `range`：用于输入数值范围，通常显示一个滑动条。
  • `search`：用于搜索框，表现上与文本框类似，但专门用于搜索输入。

3. 按钮标签 `

  • `submit`：提交表单（默认值）。
  • `reset`：重置表单字段。
  • `button`：普通按钮，可以用于绑定 JavaScript 事件处理程序。

4. HTML5 表单新增 `multiple` 属性可以用于何种类型的 `` 标签？

  • `multiple`：属性可以用于 `` 类型为 `file` 或 `email` 时，允许用户选择多个文件或输入多个电子邮件地址。

5. HTML5 表单新增 `width` 和 `height` 属性可以用于何种类型的 `` 标签？

  • `width` 和 `height`：属性可以用于 `` 类型为 `image` 时，指定提交按钮图像的大小。

6. 如何使用 HTML5 表单新增 `pattern` 属性限制用户只允许输入 6 位阿拉伯数字？

使用 `` 标签并设置 `pattern` 属性为 `\d{6}`，例如：

        
<input type="text" pattern="\d{6}" title="请输入6位阿拉伯数字">

这将要求用户输入一个恰好由 6 个数字组成的字符串。

另一种方式是使用 `[0-9]{6}` 正则表达式，效果相同：

        
<input type="text" pattern="[0-9]{6}" title="请输入6位阿拉伯数字">
